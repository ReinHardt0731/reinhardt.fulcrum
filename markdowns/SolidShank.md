# Aircraft Solid Shank Rivets Cheat Sheet

---

# Common Solid Rivet Head Styles

| Rivet | Head Style | Memory Hook | Typical Applications |
|--------|------------|-------------|----------------------|
| **AN470 / MS20470** | Universal Head | **470 = Universal** | General aircraft structures |
| **AN426 / MS20426** | Countersunk (100° Flush) | **426 = Flush** | Aircraft skin, aerodynamic surfaces |

---

# Common Rivet Materials

| Rivet Material | Identification Mark | Remember This | Typical Applications |
|----------------|---------------------|---------------|----------------------|
| **1100 (A)** | None | Pure Aluminum | Soft materials, non-structural |
| **2117 (AD)** | Dimple | **Most Common Aircraft Rivet** | General structural riveting |
| **2017 (D)** | Raised Dot | Icebox Rivet | Medium-strength structures |
| **2024 (DD)** | Two Raised Dots | Strong but Less Corrosion Resistant | High-strength structures |
| **5056 (B)** | Raised Cross | Magnesium Structures | Riveting magnesium alloys |

# Installation Rules

| Rule | Remember |
|------|----------|
| Rivet diameter | ≈ **3 × thickness of thickest sheet** |
| Rivet protrusion before driving | **1.5D** |
| Shop head diameter | **1.5D** |
| Shop head height | **0.5D** |

### Memory Hook

> **1.5 – 1.5 – 0.5 Rule**

- Stick out = **1.5D**
- Shop head diameter = **1.5D**
- Shop head height = **0.5D**

---

# Edge Distance & Rivet Spacing

| Parameter | Standard Rule |
|-----------|---------------|
| Edge Distance | **2D minimum** |
| Rivet Pitch | **3D minimum** |
| Typical Pitch | **6D–8D** |

### Memory Hook

> **2 – 3 – 6**

- Edge = **2D**
- Pitch ≥ **3D**
- Typical spacing = **6D–8D**

# Heat Treatment Requirement

| Rivet Alloy | Heat Treatment Before Use? |

| 2117 | No |
