# Composite Terminology Cheat Sheet

---

# Fiber Orientation

| Term | Meaning | Memory Hook |
|------|---------|-------------|
| **Warp** | Fibers running lengthwise (0°) | **Warp = Along the Roll** |
| **Fill (Weft)** | Fibers woven across the warp (90°) | **Fill Fills Across** |
| **Bias** | Fibers oriented at ±45° | **Bias = Diagonal** |
| **Off-Axis** | Fibers oriented at angles other than 0°, 45°, or 90° | **Not Straight** |

---

# Fabric Construction

| Term | Meaning | Typical Use |
|------|---------|-------------|
| **Plain Weave** | Simple over-under weave | General-purpose laminates |
| **Twill Weave** | Diagonal weave pattern | Better drape over curves |
| **Satin Weave** | Smooth weave with fewer interlacings | Complex contours |
| **Unidirectional (UD)** | Fibers run in one direction only | Maximum strength in one direction |

---

# Composite Layup Terms

| Term | Meaning | Memory Hook |
|------|---------|-------------|
| **Ply** | One layer of composite fabric | **One Sheet** |
| **Lamina** | Single composite layer | Same as Ply |
| **Laminate** | Multiple bonded plies | Stack of Plies |
| **Lay-up** | Arrangement of plies before curing | Building the laminate |
| **Stacking Sequence** | Order of ply orientations | Fiber Roadmap |

---

# Resin Terms

| Term | Meaning |
|------|---------|
| **Matrix** | Resin surrounding the fibers |
| **Resin-Rich Area** | Excess resin, reduced strength |
| **Resin-Starved Area** | Insufficient resin, poor bonding |
| **Prepreg** | Fiber pre-impregnated with resin |
| **Pot Life** | Working time before resin begins to cure |
| **Gel Time** | Time required for resin to reach gel stage |

---

# Sandwich Structure

| Term | Meaning |
|------|---------|
| **Face Sheet (Skin)** | Outer composite layer |
| **Core** | Lightweight center material |
| **Honeycomb Core** | Hexagonal lightweight core |
| **Foam Core** | Cellular polymer core |
| **Balsa Core** | Natural wood core |

---

# Cure Terminology

| Term | Meaning |
|------|---------|
| **Curing** | Hardening of the resin |
| **Room Temperature Cure** | Cures without external heat |
| **Oven Cure** | Uses elevated temperature |
| **Autoclave Cure** | Heat + Pressure |
| **Post Cure** | Additional heating after initial cure |

---

# Vacuum Bagging Components

| Component | Purpose |
|-----------|----------|
| **Vacuum Bag** | Applies uniform pressure |
| **Peel Ply** | Easy removal after cure |
| **Release Film** | Prevents sticking |
| **Breather Cloth** | Allows air movement |
| **Bleeder Cloth** | Absorbs excess resin |

---

# Fiber Orientation Angles

| Orientation | Purpose |
|-------------|----------|
| **0°** | Longitudinal strength |
| **90°** | Transverse strength |
| **±45°** | Shear strength |
| **Quasi-Isotropic** | Nearly equal strength in all directions |

### Memory Hook

> **0 = Pull**

> **90 = Side**

> **45 = Twist**

| **Resin Content** | Percentage of resin |
| **Void Content** | Percentage of trapped air |
