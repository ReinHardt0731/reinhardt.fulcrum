# Aircraft Composite Defects Cheat Sheet

---

# Common Composite Defects

| Defect | Description | Common Cause |
|---------|-------------|--------------|
| **Delamination** | Separation of laminate layers | Impact, overload |
| **Disbond** | Separation of adhesive bond | Poor bonding, contamination |
| **Voids** | Air pockets within laminate | Poor vacuum, improper curing |
| **Porosity** | Numerous microscopic voids | Entrapped gases |
| **Blister** | Raised surface bubble | Moisture, trapped air |
| **Wrinkles** | Folded fibers | Poor lay-up |
| **Resin Starvation** | Too little resin | Poor impregnation |
| **Resin Rich Area** | Excess resin | Poor lay-up |
| **Fiber Breakage** | Broken reinforcement fibers | Excessive impact |
| **Crushed Core** | Damaged honeycomb | Compression load |
| **Core Disbond** | Core separates from skin | Adhesive failure |
| **Foreign Object Inclusion** | Contamination trapped inside | Poor workmanship |
